前情回顾:

1. 数据基础操作

   查询:  select col,... from [tb] [where .....]

   修改:  update [tb] set  col=val,... where ....

   删除:  delete from [tb] where ...

2. 表结构修改

  alter table [tb]  add/drop/modify/change/rename

3. 时间类型

   date   datetime  time   year  timestamp

   now()  curtime()   curdate()

4. 高级查询方法

   模糊查询:  like   %  _
   正则查询:  regexp
   起名字:  as
   排序 : order by  col  [desc]
   限制: limit [n]
   联合查询: union  [all]
   子查询:   ()    from后面   where子句

5. 聚合操作

   select  ->  数据整理

   avg()   max()   min()   sum()   count()
